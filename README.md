# 4D4Y Forum CLI

一个仿早期 telnet BBS 时代界面和操作方式的 4D4Y 论坛 CLI 浏览工具。

## 快速开始

### 1. 下载代码

```bash
git clone https://github.com/gigode/4d4y_cli.git
cd 4d4y_cli
```

### 2. 安装依赖

```bash
# 创建虚拟环境（推荐）
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate      # Linux/Mac
venv\Scripts\activate         # Windows

# 安装依赖
pip install requests beautifulsoup4
```

### 3. 运行程序

```bash
python -m forzd4y.cli
```

## 使用说明

### 首次使用

1. 运行程序后，界面会显示主菜单
2. 按 `1` 浏览论坛板块，找到要进入的板块编号
3. 按 `L` 登录论坛（需要注册账号）
4. 登录后才能访问 Discovery 板块和回复帖子

### 主菜单操作

| 按键 | 功能 |
|------|------|
| `1` + 回车 | 浏览论坛板块列表 |
| `2` + 回车 | 直接进入 Discovery 板块（需登录） |
| `L` + 回车 | 登录 / 注销 |
| `Q` + 回车 | 退出程序 |

### 板块内浏览帖子

进入板块后，会显示帖子列表：

| 按键 | 功能 |
|------|------|
| `↑` / `↓` 或 `J` / `K` | 上/下移动选择帖子（带光标▶显示） |
| `Enter` | 打开选中的帖子 |
| `R` + 回车 | 回复当前选中的帖子（需登录） |
| `B` + 回车 | 返回板块列表 |
| `Q` + 回车 | 退出程序 |

### 阅读帖子内容

| 按键 | 功能 |
|------|------|
| `J` 或 `↓` | 下一页 |
| `K` 或 `↑` | 上一页 |
| `R` + 回车 | 回复此帖（需登录） |
| `H` + 回车 | 跳到第一页 |
| `L` + 回车 | 跳到最后一页 |
| `1-9` + 回车 | 跳转到指定页 |
| `Q` + 回车 | 返回帖子列表 |

### 回复帖子

1. 在帖子列表选中要回复的帖子，按 `R` + 回车
2. 或进入帖子内容页面，按 `R` + 回车
3. 输入回复内容
4. 输入完成后按 `Ctrl+D`（或空行后回车）发送
5. 按 `Ctrl+C` 取消回复

## 按键说明

### 光标移动
- `↑` / `↓` - 上下箭头键
- `J` / `K` - Vim 风格键位（J=下，K=上）

### 确认与取消
- `Enter` - 确认选择
- `Ctrl+C` - 取消当前操作
- `Ctrl+D` - 结束输入（发送回复）

### 快速退出
- `Q` - 在任何界面按 Q 都可以快速退出程序

## 配置文件

登录信息保存在用户配置目录下：

- Linux/Mac: `~/.config/4d4y_cli/`
- Windows: `C:\Users\用户名\AppData\Local\4d4y_cli\`

包含文件：
- `config.json` - 用户名等配置
- `cookies.json` - 登录会话

如需重新登录，清除这些文件即可。

## 常见问题

### Q: 为什么显示"未登录"？
A: 需要按 `L` 输入用户名密码登录后才能访问 Discovery 和回复帖子。

### Q: 帖子中的图片怎么显示？
A: 图片以链接形式显示，不会下载显示。复制链接到浏览器可以查看。

### Q: 如何退出程序？
A: 在任何界面按 `Q` 即可退出。
