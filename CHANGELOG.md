# Changelog

## 1.0.2 - 2026-04-12

### 交互调整
- 全项目统一调整 Vim 风格导航键位：`J` 现在表示向上，`K` 现在表示向下。
- 同步更新帖子列表、帖子阅读页、交互选择器和界面提示文案，避免不同页面行为不一致。

### 帖子阅读增强
- 帖子阅读页中的回帖超链接现在会和 JPG 图片一样显示为可选项。
- 可使用 `J/K` 或方向键上下选中图片和超链接，并按 `Enter` 调用系统默认浏览器打开。
- 帖子解析阶段会提取回帖中的普通链接，同时跳过图片链接、`javascript:` 链接和重复链接。

### 文档与测试
- 更新 README 中的按键说明、阅读页说明和常见问题，反映新的键位和链接打开行为。
- 新增回帖超链接提取与打开行为测试，并同步调整现有图片选择测试。

## 1.0.1 - 2026-04-10

### 修复
- 修复密码错误时仍提示登录成功的问题。
- 收紧登录状态判定，并在登录失败时清理本地无效登录状态。
- 增加登录失败场景的回归测试。

## 1.0.0 - 2026-04-09

- 发布 4D4Y Forum CLI 的首个稳定版本。
