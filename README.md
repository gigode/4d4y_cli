# 4D4Y Forum CLI

一个仿早期 telnet BBS 时代界面和操作方式的 4D4Y 论坛 CLI 浏览工具。

## 快速开始

### 1. 下载代码

```bash
git clone https://github.com/gigode/4d4y_cli.git
cd 4d4y_cli
```

### 2. 一键安装

```bash
./4d4yinstall
```

安装程序会自动：
- 创建 Python 虚拟环境
- 安装依赖 (requests, beautifulsoup4)
- 创建 `4d4y` 快捷命令

安装完成后，请确保 `~/.local/bin` 在你的 PATH 中：
```bash
export PATH="$HOME/.local/bin:$PATH"
```

### 3. 运行程序

```bash
./4d4y
```

或如果已将 `~/.local/bin` 加入 PATH：
```bash
4d4y
```

## 使用说明

### 首次使用

1. 运行程序后，界面会显示主菜单
2. 按 `1` 浏览论坛板块，找到要进入的板块编号
3. 按 `L` 登录论坛（需要注册账号）
4. 登录后才能访问 Discovery 板块和回复帖子

### 主菜单操作

| 按键 | 功能 |
|------|------|
| `1` | 浏览论坛板块列表 |
| `2` | 直接进入 Discovery 板块（需登录） |
| `L` | 登录 / 注销 |
| `Q` | 退出程序 |

### 板块内浏览帖子

进入板块后，会显示帖子列表：

| 按键 | 功能 |
|------|------|
| `↑` / `↓` 或 `J` / `K` | 上/下移动选择帖子（带光标▶显示） |
| `H` / `L` | 上一页 / 下一页 |
| `Enter` | 打开选中的帖子 |
| `1-20` + 回车 | 直接进入当前页对应编号的帖子 |
| `R` | 刷新当前页 |
| `B` | 返回板块列表 |
| `Q` | 退出程序 |

帖子列表每页显示 20 个帖子。

### 阅读帖子内容

| 按键 | 功能 |
|------|------|
| `J` / `K` 或 `↑` / `↓` | 在当前页回复之间上下选择 |
| `Enter` | 选中图片时在终端内查看 |
| `H` / `L` | 上一页 / 下一页 |
| `R` | 刷新当前帖子内容 |
| `B` | 返回帖子列表 |
| `1-9` + 回车 | 跳转到指定页 |
| `Q` | 退出程序 |

帖子阅读界面按终端实际可见高度分页，正文部分最多显示 20 行；如果当前终端较矮，会进一步压缩，优先保证单页不需要滚动。长回复会自动拆到后续页继续显示。
如果帖子里有图片，会在正文下方显示为可选图片项；选中后按 `Enter` 会用系统默认浏览器打开。

### 回复帖子

1. 在帖子列表选中要回复的帖子，按 `R`
2. 或进入帖子内容页面，按 `R`
3. 输入回复内容
4. 输入完成后按 `Ctrl+D`（或空行后回车）发送
5. 按 `Ctrl+C` 取消回复

## 按键说明

### 光标移动
- `↑` / `↓` - 上下箭头键
- `J` / `K` - Vim 风格键位（J=下，K=上）
- `H` / `L` - 上一页 / 下一页

### 确认与取消
- `Enter` - 用于数字编号或页码输入后的确认
- `Ctrl+C` - 取消当前操作
- `Ctrl+D` - 结束输入（发送回复）

### 快速退出
- `Q` - 在主菜单、板块列表、帖子列表、帖子阅读界面都可直接退出程序
- `B` - 在板块列表、帖子列表、帖子阅读界面返回上一级

## 配置文件

登录信息保存在用户配置目录下：

- Linux/Mac: `~/.config/4d4y_cli/`
- Windows: `C:\Users\用户名\AppData\Local\4d4y_cli\`

包含文件：
- `config.json` - 用户名等配置
- `cookies.json` - 登录会话

如需重新登录，清除这些文件即可。

## 常见问题

### Q: 为什么显示"未登录"？
A: 需要按 `L` 输入用户名密码登录后才能访问 Discovery 和回复帖子。

### Q: 帖子中的图片怎么显示？
A: 图片以链接形式显示，不会下载显示。复制链接到浏览器可以查看。

### Q: 如何退出程序？
A: 在主菜单、板块列表、帖子列表、帖子阅读界面按 `Q` 即可退出；按 `B` 返回上一级。

## 系统要求

- Python 3.7 或更高版本

- `requests` - HTTP 请求库
- `beautifulsoup4` - HTML 解析库
